## Installing

1. Download the code
5. Run the following command:
```
    npm install
```
6. Run the following commnad to start the node server (default port 8000):
```
    node server.js
```
---
## Available APIs

**MongoDB**
* GET    /api/kudos/list
* POST   /api/kudos/add
* DELETE /api/kudos/del/{id}
